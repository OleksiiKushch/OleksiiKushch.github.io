Project Information

  • Internal project code: EPM-HYTC
  • Work period: Sep, 2022 - Mar, 2024 (1 year 7 months)
  • Industry: Retail & Distribution

Project Description

Internal Hybris Training Project

The primary objective of the project is to develop a comprehensive e-commerce platform that provides a diverse range of services to consumers. This involves not only facilitating and streamlining the entire online shopping experience for customers but also extending related functionalities to boost sales.

E-commerce website built on SAP Commerce platform (Hybris) as the core system.

Project Role(s)

Engineer (Hybris Developer)

Responsibilities

  • Create robust architectural solutions and write clean, efficient, and maintainable code according to the project's coding standards;
  • Develop unit tests, integration tests, and BDD (Cucumber) tests to uphold code quality. Perform functional testing, including manual testing, on existing features to identify and report bugs and issues;
  • Participate in code reviews to provide feedback and maintain code quality. Perform code refactoring, maintain, and enhance the existing codebase, repay technical debt;
  • Provide regular updates to team members during daily meetings (current task progress, challenges, and solutions effectively);
  • Investigate and document new/old features, write guides and tutorials (increase the project's knowledge base and facilitate knowledge sharing among developers).

Additional Info

Achievements

Implementing the following functionality:

  • product comparison;
  • customer anonymization;
  • recommended product;
  • customer reaction to review;
  • customer product questions;
  • registration with referral code;

Integration Cucumber for BDD tests into the project.

Updating the Hybris version in the project (from 1905 to 2005 and then to 2011).

Made some improvements to the loyalty/bonus, premium, and achievement internal systems.
